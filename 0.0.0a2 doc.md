# 操作说明：
## 如何移动鸭妈妈 
程序开始后，鸭妈妈会跟着鼠标移动，他会在不突破格子的情况下尽量跟随鼠标。

## 如何收集小鸭子 
鸭妈妈靠近小鸭子的时候，小鸭子就会跟随着妈妈直到靠近鸭子窝。
